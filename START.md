# 🚀 快速启动指南

## 一键启动（推荐）

### macOS 用户

**双击运行**：
```
启动服务.command
```

或者在终端中运行：
```bash
./run.sh
```

### 其他系统

在终端中运行：
```bash
./run.sh
```

## 使用步骤

### 1️⃣ 启动服务

运行启动脚本后，您会看到：

```

║                                                        ║
║        🎨 PixelPerfect Type 字体验收工具 🎨          ║
║                                                        ║
╚════════════════════════════════════════════════════════╝

✅ Python 版本: 3.x.x

🚀 正在启动服务...

============================================================
  🎨 PixelPerfect Type - 字体验收工具
============================================================
✨ 服务已启动！

📍 访问地址: http://localhost:9090

💡 使用说明:
   1. 在浏览器中打开上面的地址
   2. 上传UI设计稿（JPG/PNG格式）
   3. 点击'开始分析'按钮
   4. 查看四个视图的分析结果
```

### 2️⃣ 打开网站

在浏览器中访问：

**👉 http://localhost:9090**

### 3️⃣ 开始使用

1. 点击"选择文件"或拖拽图片到上传区域
2. 点击"开始分析"按钮
3. 等待10-30秒
4. 查看四个视图的分析结果：
   - View 1: 图像标准化
   - View 2: OCR文字识别
   - View 3: 字号拟合
   - View 4: 结果标注

**PaddleOCR 模型下载**

首次运行时，系统会自动下载 OCR 模型文件（约100MB），请：
- ✅ 保持网络连接
- ✅ 耐心等待5-10分钟
- ✅ 只需下载一次

## 🆘 遇到问题？

### 问题1: 端口被占用

如果看到 "Address already in use" 错误：

```bash
# 查找占用5000端口的进程
lsof -i :5000

# 杀死进程（替换PID为实际进程号）
kill -9 PID
```

### 问题2: Python未安装

请访问 https://www.python.org/downloads/ 下载安装 Python 3.8+

### 问题3: 依赖安装失败

```bash
# 手动安装依赖
cd backend
pip install -r requirements.txt
```
